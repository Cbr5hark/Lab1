    Інструкція розв'язування квадратного рівняння. Для прикладу розв'яжено двадратне рівняння x2 + 4x - 5 = 0
  1. Знаходимо коефіцієнти a, b, c: a = 1; b = 4; c = -5
  2. Знаходимо дискримінант за формулою D = b2 - 4*a*c: 42 - 4 * 1 * (-5) = 36
    3. Якщо дискримінант від'ємний, то рівняння не має розв'язків, якщо дискримінант = 0, то рівняння має лише один розв'язок, якщо дискримінант додатній, то рівняння має 2 розв'язки
  3. У нашому випадку D > 0, тому шукаємо розв'язки за формулами: x1 = (-b + sqrt(D))/2; x2 = (-b - sqrt(D))/2: x1 = 1; x2 = -5
